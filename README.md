# Luciano's dotfiles

My personal dotfiles repo

## Installation

Take the files you want and put them in your `~` folder

## Arch/voidrice

To overwrite `voidrice` files, take the following files:

* .generalrc
* .zshrc
* .bashrc
* .vimrc
* .vim folder (merge with existing files)
* .oh-my-zsh folder
